Microcontroller-Based Smart Assistive Dumbbell with Software Application

Date Accomplished: April 26, 2023


Role: Software Application Developer


Language/s: Arduino, Kivy, Firebase


Description: Thesis project. Arduino microcontroller is integrated to a dumbbell to perform certain actions. The Smart Dumbbell can count every repetition, detects if the orientation is correct and notifies the user if the dumbbell is properly lowered. The Smart Dumbbell is connected to a software application via Bluetooth and displays every repetition that the user has executed. After every workout, the number of repetitions that the user executed is saved and can be viewed in their accounts. The software application allows users to create an account so they can view their progress every workout.

System Health Monitoring

Date Accomplished: July 20, 2023


Role: Developer


Language/s: C#


Description: A collaborative internship project. The System Health Monitoring have five (5) features. First is the "CPU and Memory" wherein it displays the performance of the CPU and memory of the computer used. Second is the "Web Applications". Users can add, edit and delete a list of web applications and the application will tell whether the web application added is accessible or not. Third is "Files". Users can add, edit and delete a certain files from their computer and set a file limit so that the application will notify whether the file name added is exceeding the limit. Fourth is "Services". Users can add, edit and delete certain services to check whether it is running or not. Last is the "Schedule". This allows users to schedule the monitoring of "Web Applications", "Files" and "Services" and the results will be sent via Email.
